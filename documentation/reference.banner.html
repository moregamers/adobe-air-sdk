MoreGamers.banner(target:DisplayObjectContainer, x:Number, y:Number, width:Number, height:Number);

Add a MoreGamers banner to the display.

Parameters

target
The parent object that you would call addChild() on.
x
The x position of the banner relative to target.
y
The y position of the banner relative to the target
width
The width of the banner area
height
The height of the banner area

Examples

MoreGamers.banner(stage, 0, 0, 525, 145);
