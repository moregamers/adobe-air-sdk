Installation

1. Copy MoreGamers.as into the source directory of your game.
2. Call MoreGamers.init(GAME_ID) to initialize the banner component. Your GAME_ID can be found at http://app.moregamers.com. You only need to call init() once. 
MoreGamers.init(1);
3. Call MoreGamers.banner(target, x, y, width, height) to display a banner. The target is the parent object that you would call addChild() on. The x and y values are relative to the parent. 
MoreGamers.banner(this, 0, 0, 525, 145);
